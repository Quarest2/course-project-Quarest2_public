
## Элементы DFD

### Процессы
1. **Feature Votes API** - Основное приложение FastAPI
   - Обработка REST API запросов
   - Бизнес-логика голосования
   - Валидация данных

### Внешние сущности
- **Пользователь** - конечный пользователь системы
- **Админ панель** - интерфейс для администраторов
- **База данных** - хранилище данных (SQLite/Postgres)

### Потоки данных (Flows)

| ID  | Направление | Данные | Протокол | Примечания |
|-----|-------------|--------|----------|------------|
| F1  | User → API  | HTTP запросы (GET/POST) | HTTP/HTTPS | Создание фич, голосование |
| F2  | API → DB    | SQL запросы | TCP | CRUD операции |
| F3  | API → User  | HTTP ответы | HTTP/HTTPS | JSON данные |
| F4  | DB → API    | Результаты запросов | TCP | Данные из БД |
| F5  | Admin → API | Админские запросы | HTTP/HTTPS | Управление системой |

## Границы доверия

### Trust Boundary 1: Интернет ↔ Наш сервер
- Потоки: F1, F3, F5
- Риски: MITM, инъекции, подделка запросов

### Trust Boundary 2: Приложение ↔ База данных
- Потоки: F2, F4
- Риски: SQL инъекции, несанкционированный доступ

## Компоненты системы

- **Web Server**: FastAPI + Uvicorn
- **Database**: SQLite (dev) / Postgres (prod)
