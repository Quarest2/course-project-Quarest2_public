# Threat Model - Feature Votes API

## Обзор
Данная модель угроз описывает потенциальные угрозы безопасности для системы голосования за фичи.

## Структура
- [DFD Diagram](MJ%20DFD.md) - Data Flow Diagram с границами доверия
- [STRIDE Analysis](MJ%20STRIDE.md) - Анализ угроз по методике STRIDE  
- [Risk Registry](MJ%20RISKS.md) - Реестр рисков и план обработки
- [NFR Requirements](NFR_REQUIREMENTS.md) - Нефункциональные требования безопасности
- [NFR BDD Scenarios](NFR_BDD_SCENARIOS.md) - BDD сценарии для верификации NFR
- [NFR Traceability](NFR_TRACEABILITY.md) - Матрица трассировки требований

## Компоненты системы
- **Frontend**: Web интерфейс для голосования
- **API Gateway**: Маршрутизация запросов
- **Vote Service**: Обработка голосов и результатов
- **Database**: Хранение данных голосования
- **Authentication Service**: Управление доступом

## Ключевые угрозы
- Несанкционированное голосование
- Манипуляция результатами
- Отказ в обслуживании
- Утечка данных пользователей

## NFR Безопасность
Система соответствует 8 нефункциональным требованиям безопасности (NFR-001 - NFR-008), включая производительность, доступность, защиту данных и масштабируемость.
